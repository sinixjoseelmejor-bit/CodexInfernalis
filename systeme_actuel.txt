1. BOUCLE DE JEU
═══════════════════════════════════════════════════════

Le joueur traverse des vagues de 40 secondes (niveaux 1 à 6), puis affronte un boss au niveau 7.
Entre chaque vague, un écran de shop permet d'acheter des items avec des clés et d'améliorer
ses stats avec des âmes. Une run complète demande 7 niveaux.

Les âmes gagnées en run sont conservées entre les runs pour upgrader les stats.
Les items sont réinitialisés à chaque run.
Les skills permanents (Forge Éternelle) sont débloqués avec des âmes de boss (drop du boss).


═══════════════════════════════════════════════════════
2. STATS DU JOUEUR
═══════════════════════════════════════════════════════

STATS DE BASE (sans aucun upgrade ni item) :
  HP max      : 5
  Dégâts      : 1
  Vitesse     : 275 px/s
  Cadence tir : 0.10s par tir (actuellement — voir note)

NOTE IMPORTANT : la cadence de base a été modifiée à 0.10s (10 tirs/s) mais le plancher
est toujours à 0.25s, ce qui rend les upgrades de cadence inutiles. À corriger.

FORMULE DE CALCUL FINALE :
  HP max  = (5 + lvl_hp×2 + flat_hp) × (1 + pct_hp)
  Dégâts  = (1 + lvl_damage + flat_dmg) × (1 + pct_dmg)
  Vitesse = (275 + lvl_speed×20 + flat_spd) × (1 + pct_spd)
  Cadence = max(0.25, (base_cd + flat_cd) × (1 - pct_cd))

UPGRADES DE RUN (shop entre les vagues) :
  Coûts : 10 → 25 → 50 → 100 → 200 âmes (5 niveaux max)
  HP      : +2 HP max par niveau
  Dégâts  : +1 dégât par niveau
  Vitesse : +20 px/s par niveau
  Cadence : -0.09s par niveau

INVINCIBILITÉ APRÈS COUP :
  Durée de base : 0.5s
  Bonus phantom_step : +1.0s par stack


═══════════════════════════════════════════════════════
3. SYSTÈME DE TIR DU JOUEUR
═══════════════════════════════════════════════════════

COMPORTEMENT :
  - Tir automatique vers l'ennemi le plus proche dans un rayon de 400px
  - Si aucun ennemi dans le rayon : le joueur ne tire pas
  - Clic souris : override l'auto-aim vers le curseur pendant 2 secondes
  - Les ennemis morts (dead = true) sont ignorés par l'auto-aim même avant leur disparition

PROJECTILE (Fireball) :
  Vitesse de base   : 1200 px/s
  Vitesse vélocité  : 2020 px/s (skill Forge Éternelle)
  Portée max        : 400 px (auto-destruction)
  Pénétration       : 1 ennemi par défaut, 3 avec skill Pénétration
  Dégâts            : PlayerData.damage (modifié par enrage)

DÉGÂTS EFFECTIFS :
  Normal  : PlayerData.damage × 1.0
  Enragé  : PlayerData.damage × 1.5 (pendant 2s après un kill, si rage_ring)

AFFICHAGE DU RAYON (debug) :
  Maintenir ALT : affiche le cercle d'auto-aim en bleu translucide

4. COMPÉTENCES ACTIVES PASSIVES DU JOUEUR
═══════════════════════════════════════════════════════

Ces effets se déclenchent automatiquement selon les items/skills possédés :

  fire_boots (item rare)
    → Traînée de feu toutes les 0.15s quand le joueur se déplace
    → Chaque traînée dure 2.0s, inflige 1 dégât toutes les 0.5s
    → ~4 dégâts max si un ennemi reste dedans

  auto_grenade (item épique)
    → Grenade lancée automatiquement toutes les 8.0s
    → Cible l'ennemi le plus proche, ou position aléatoire (±300px)
    → Explose après 0.9s, rayon 140px, 3 dégâts

  storm_ring (item épique)
    → Salve de 8 projectiles omnidirectionnels toutes les 15.0s
    → Se stack : 2 anneaux = 16 projectiles par salve

  tempete_acier (skill Forge Éternelle — final)
    → Salve de 12 projectiles omnidirectionnels toutes les 10.0s

  thorn_shield (item rare)
    → Renvoie ceil(dégât × 15% × stacks) à tous les ennemis dans un rayon de 280px

  vampire_amulet (item rare)
    → Soigne (dégâts_infligés × 1% × stacks) — accumulé jusqu'à 1 HP entier

  phantom_step (item rare)
    → +1.0s d'invincibilité par stack après chaque coup reçu

  rage_ring (item rare)
    → Après chaque kill ennemi : dégâts ×1.5 pendant 2.0s

9. ÉCONOMIE — CLÉS ET ÂMES
═══════════════════════════════════════════════════════

CLÉS :
  Source principale : drops d'Aldrich
  Drop rate          : 17% au niv.1, décroit jusqu'à 3% au niv.6
  Utilisation        : ouvrir des caisses au shop

CAISSES (shop) :
  Commun  : 1 clé,  60% de chance d'apparaître, 10 items dans le pool
  Rare    : 2 clés, 30% de chance,               5 items dans le pool
  Épique  : 3 clés, 10% de chance,               3 items dans le pool
  Reroll  : 30 âmes (+50 à chaque reroll suivant), ne touche pas les caisses verrouillées

ÂMES DE RUN :
  Source   : 1 âme par kill ennemi (+1 par soul_harvester possédé)
  Usage    : Upgrade stats au shop (10→25→50→100→200 par niveau)
  Persistance : gardées entre les runs

ÂMES DE BOSS :
  Source   : drop de Golgota à sa mort
  Usage    : déverrouiller des nœuds dans la Forge Éternelle (1 âme/nœud)
  Persistance : permanentes

11. SAUVEGARDE
═══════════════════════════════════════════════════════

Fichier : user://save.json
Données persistantes sauvegardées :
  - boss_souls    : âmes de boss accumulées
  - kills_total   : total de kills depuis le début
  - perm_skills   : skills Forge Éternelle débloqués par personnage
  - âmes de run   : NON sauvegardées (reset à chaque run)
  - items         : NON sauvegardés (reset après boss kill)
  - niveaux upgrade: NON sauvegardés (reset chaque run)
